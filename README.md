This is a readme for the script to make a xyz file. 
KAG created this script with the help of AI (ChatGPT IIRC) and implemented it using Python 3.9 via IDLE.

The script helps speed up data processing for DFT calculations where you need to make a .xyz file of all your optimized geometries. Prior to this, I would open each .log file from Guassian in GaussView, then save as .mol2, then use Mercury to save as .xyz, then use notepad++ to copy and past the xyz coords into a single .xyz file. It is annoying, so why not automate it? 

I used Chatgpt to make the script.

I am sure it could be done better (I don't know how to code), but it's worked for me!
